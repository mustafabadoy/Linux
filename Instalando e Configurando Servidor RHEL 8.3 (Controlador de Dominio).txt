Instalando e Configurando o Servidor Red Hat Linux Enterprise 8.3

1- Esquema de Particionamento Personalizado em LVM

Partição Raiz (/) Tipo de Dispositivo: (LVM) Sistema de Arquivos: (XFS)
Partição Arquivos Temporários (/tmp) Tipo de Disposito: (LVM) Sistema de Arquivos (XFS) - 200MB
Partição Arquivos de Log (/var) Tipo de Disposito: (LVM) Sistema de Arquivos (XFS) - 500MB
Partição Arquivos de Usuários(/home) Tipo de Disposito: (LVM) Sistema de Arquivos (XFS)
Partição Arquivos Inicialização (/boot) Tipo de Disposito: (Partição Padrão) Sistema de Arquivos (XFS) - 350MB
Partição Arquivos de Troca (/swap) Tipo de Disposito: (LVM) Sistema de Arquivos (swap) 2GB

2- Registrando o RHEL

#subscription-manager register --username badoy --password 123456

Verificando o Registro
#subscription-manager attach --auto

3- Especificando nome do servidor (server)

#hostnamectl set-hostname server
Verificando:
#hostname

4- Específicando nome do dominio (empresa.local)

Editar o arquivo: hosts 
#/etc
Comentar 2ª linha 
Inserir Linha: 192.168.0.100 server.empresa.local server
Verificando:
#hostname -f

5- Confiruação das Placas de Rede

Listar IP das Placa de Rede
#ifconfig
#ip a

Editar os arquivos: ifcfg-ens160, ifcfg-ens192
#/etc/sysconfig/network-scripts

Editar arquivo: resolv.conf
#/etc

search empresa.local
nameserver 192.168.0.100
nameserver 8.8.8.8

6- Travar o NetworkManager

#chattr +i /etc/resolv.conf
#chattr +i /etc/resolv.conf (Destrava o NetworkManager)


Considerações sobre FirewallD

#systemctl status firewalld (Verificar Status)

#firewall-cmd --help (Help)

#firewall-cmd --list-all (Lista serviços ativos)

#systemctl disable firewalld (Desabilitar Firewall)

Considerações Sobre Selinux

#getenforce (Verificar parametros do Selinux)

#setenforce (Mostra parametros do Selinux)

#setenforce 1 (Muda parametro para: Enforcing)

#setenforce 0 (Muda parametro para: Permissive)

Desativando a Interface Gráfica:

ln -sf /usr/lib/systemd/system/multi-user.target /etc/systemd/system/default.target

Se quiser reabilitar o modo gráfico:

ln -sf /usr/lib/systemd/system/graphical.target  /etc/systemd/system/default.target
